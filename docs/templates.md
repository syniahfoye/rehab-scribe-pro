# Rehab Nursing Assessment Templates

## Scope

Template set optimized for rehab clinic documentation burden reduction.

## Required template family

1. Initial nursing assessment
2. Daily progress assessment
3. Functional status reassessment
4. Pain and symptom tracking note
5. Discharge readiness assessment

## MVP template implemented

- `rehab_nursing_assessment_v1`
- Required fields are marked in `shared/templates.ts`
- Fields are grouped into patient overview, functional status, and clinical assessment

## Clinical stakeholder validation checklist

- Confirm required fields match clinic policy
- Confirm language aligns with local documentation standards
- Confirm export format aligns with EHR intake expectations
