# Pilot and Integration Plan

## Pilot objectives

- Reduce documentation time by at least 30%
- Achieve at least 90% acceptance of drafted sections with minor edits
- Ensure 100% clinician sign-off before finalization

## Pilot phases

1. Internal dry-run with synthetic encounters
2. Limited clinical pilot (2-3 nurses, rehab clinic)
3. Iteration based on edits, missing fields, and confidence trends

## Integration path

- Start with JSON/PDF export
- Add FHIR `DocumentReference` export adapter
- Add HL7 adapter only if required by target EHR
